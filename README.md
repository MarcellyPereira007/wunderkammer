### System Log
> **Gabinete de curiosidades digital:** Um projeto autoral focado na curadoria de mídias, hiperfocos e exploração proativa. 
> O Wunderkammer não é só um site, é um repositório de experiências personalizadas.

### Stack
| Componente | Tecnologia |
| :--- | :--- |
| **Frontend** | HTML5, CSS3, JavaScript |
| **Backend** | Node.js, Express |
| **Banco de dados** | MySQL (Hospedado em VM) |
| **Estética** | Y2K, Webcore, ASCII Art, Old Web |

### Comandos de inicialização
Para rodar o sistema localmente, execute no terminal:
